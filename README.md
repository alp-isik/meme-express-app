# README.md for meme-express-app

# Meme Express App

A simple web application built with Node.js and Express that allows users to log in and browse memes from an external meme API. The project focuses on basic backend logic, routing, authentication flow, and server-side rendering using EJS.

## Features
- Login system with basic authentication logic
- Meme browsing from external API
- Server-side rendering with EJS
- Express routing structure
- Static assets and views separation

## Tech Stack
- Node.js
- Express
- EJS
- JavaScript
- HTML & CSS

## Project Purpose
This project was built as a school assignment during my backend development studies. The goal was to practice:

- Express server setup
- Routing and middleware
- Basic authentication logic
- Working with external APIs
- Server-side rendering

## How to Run

1. Clone the repository
2. Open the project folder in your terminal
3. Install dependencies:

   npm install

4. Start the server:

   npm start

5. Open your browser and go to:

   http://localhost:3000

## Notes
This is a learning project focused on backend fundamentals and structure rather than production-level security or architecture.
